[english] How can I programmatically identify which positional arguments of a function or class are strictly required when initializing it?
[english] This approach uses argument inspection to analyze the signature of the provided callable object.
<switch:python>
[python] def _get_required_args(fn):
  """Returns the distribution's required args."""
  argspec = tf_inspect.getfullargspec(fn)
  args = argspec.args
  if tf_inspect.isclass(fn):
    args = args[1:]  # Remove the `self` arg.
  if argspec.defaults:
    # Remove the args which have defaults. By convention we only feed
    # *required args*. This means some distributions must always be wrapped
    # with a `lambda`, e.g., `lambda logits: tfd.Bernoulli(logits=logits)`
    # or `lambda probs: tfd.Bernoulli(probs=probs)`.
    args = args[:-len(argspec.defaults)]
  return tuple(args)

[english] The function systematically strips away `self` and any parameters that have default values, leaving only truly mandatory inputs.
[english] What happens if the input function has keyword-only arguments defined?
[english] This implementation focuses solely on positional arguments and their defaults. It will not capture arguments specified using keywords after *args.
