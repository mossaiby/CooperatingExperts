[english] How can I automatically decode all base64-encoded PNG and JPEG images stored in my notebook outputs?
[english] To restore image data, this function traverses every worksheet and cell within the notebook. It specifically looks for encoded strings associated with PNG or JPEG file types and processes them byte by byte.
<switch:python>
[python] def base64_decode(nb):
    """Restore all bytes objects in the notebook from base64-encoded strings.
    
    Note: This is never used
    """
    for ws in nb.worksheets:
        for cell in ws.cells:
            if cell.cell_type == 'code':
                for output in cell.outputs:
                    if 'png' in output:
                        if isinstance(output.png, unicode):
                            output.png = output.png.encode('ascii')
                        output.png = decodestring(output.png)
                    if 'jpeg' in output:
                        if isinstance(output.jpeg, unicode):
                            output.jpeg = output.jpeg.encode('ascii')
                        output.jpeg = decodestring(output.jpeg)
    return nb
