[english] How can I ensure my database is correctly configured with essential extensions like 'hstore' before running migrations?
[english] This method prepares the database environment by attempting to enable specific PostgreSQL extensions using a dedicated cursor. It uses error handling to manage situations where automatic creation might fail due to permissions.
<switch:python>
[python] def prepare_database(self):
        """Ran to prepare the configured database.

        This is where we enable the `hstore` extension
        if it wasn't enabled yet."""

        super().prepare_database()
        with self.cursor() as cursor:
            try:
                cursor.execute('CREATE EXTENSION IF NOT EXISTS hstore')
            except ProgrammingError:  # permission denied
                logger.warning(
                    'Failed to create "hstore" extension. '
                    'Tables with hstore columns may fail to migrate. '
                    'If hstore is needed, make sure you are connected '
                    'to the database as a superuser '
                    'or add the extension manually.',
                    exc_info=True)
