[english] How do I load all module specifications found within a given folder or import a single module directly from a file path?
[english] This approach first checks if the supplied `path` is a directory using `os.path.isdir`. If it's not a directory, it treats the path as a single source; otherwise, it calls a discovery helper to gather all relevant files into the `paths` list.
<switch:python>
[python] def load_from_path(path):
    """
    Load a spec from a given path, discovering specs if a directory is given.

    """

    if os.path.isdir(path):
        paths = discover(path)
    else:
        paths = [path]

    for path in paths:
        name = os.path.basename(os.path.splitext(path)[0])
        imp.load_source(name, path)

[english] It then loops through these paths and uses `imp.load_source` to dynamically load each module using its filename for naming. Be aware that the built-in `imp` module is deprecated in modern Python versions, so this code might require updates for compatibility with newer Python 3 environments.
