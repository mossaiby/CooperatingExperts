[english] How can I swap the positions of two specific dimensions within a multi-dimensional array?
[english] To achieve this, we first generate a list representing all existing axes. We then modify this index list by exchanging the values corresponding to the two target axes before applying the standard transpose operation.
<switch:python>
[python] def swapaxes(self, axis1, axis2):
        """
        Return the array with two axes interchanged.

        Parameters
        ----------
        axis1 : int
            The first axis to swap

        axis2 : int
            The second axis to swap
        """
        p = list(range(self.ndim))
        p[axis1] = axis2
        p[axis2] = axis1

        return self.transpose(p)
<switch:english>
[english] This method efficiently reorders dimensions by passing a carefully constructed permutation to the `transpose` function. This pattern is very common in libraries like NumPy and PyTorch for data manipulation.
