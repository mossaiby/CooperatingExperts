[english] I need a robust way to add a new menu item dynamically, ensuring it is placed correctly and the visual interface is resized if necessary.
[english] This method manages dynamic layout changes by first temporarily removing the 'exit' option. It adds the new item, restores the exit logic, and checks if the display needs resizing based on the current number of items.

[python] def append_item(self, item):
        """
        Add an item to the end of the menu before the exit item

        :param MenuItem item: The item to be added
        """
        did_remove = self.remove_exit()
        item.menu = self
        self.items.append(item)
        if did_remove:
            self.add_exit()
        if self.screen:
            max_row, max_cols = self.screen.getmaxyx()
            if max_row < 6 + len(self.items):
                self.screen.resize(6 + len(self.items), max_cols)
            self.draw()
