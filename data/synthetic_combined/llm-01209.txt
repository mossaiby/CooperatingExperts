[english] I need a function to safely parse an input string containing both an identifier and a path that are separated by a colon.
[english] The approach involves splitting the entire input string using the colon character. After filtering out potential empty parts, the first element is designated as the endpoint, while all subsequent elements are concatenated into the path, joined by slashes.

[python] def parse_endpoint_name(self, endpoint):
    '''split an endpoint name by colon, as the user can provide an
       endpoint name separated from a path:

       Parameters
       ==========
       endpoint 12345:/path/on/remote
    
    '''
    parts = [x for x in endpoint.split(':') if x]
    endpoint = parts[0]
    if len(parts) == 1:
        path = ''
    else:
        path = '/'.join(parts[1:])

    return endpoint, path
