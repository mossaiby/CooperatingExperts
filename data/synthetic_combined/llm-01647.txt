[english] To account for potential NaNs, we first use np.nanstd to get the deviation across valid data points and then normalize this by the square root of the count of finite elements.
<switch:python>
[python] def stderr(a):
    """
    Calculate the standard error of a.
    """
    return np.nanstd(a) / np.sqrt(sum(np.isfinite(a)))
<switch:english>
[english] This design ensures that the population standard error is calculated using only the reliably measured samples in the dataset.
