[english] To determine the names of an argument that receives some value, we first retrieve the function's full argument specification using `get_cached_arg_spec`. Then, by comparing the length of the received positional arguments (`args`) against all available named parameters in the spec, we can extract only those needed.

[python] def _get_supplied_positional_parameter_names(fn, args):
  """Returns the names of the supplied arguments to the given function."""
  arg_spec = _get_cached_arg_spec(fn)
  # May be shorter than len(args) if args contains vararg (*args) arguments.
  return arg_spec.args[:len(args)]
