[english] I am building a document modification tool and need a way to insert chunks of text marked for deletion into my main document list.
[english] To mark these deletions, we will wrap the provided `del_chunks` within predefined markers (`DEL_START` and `DEL_END`) before adding them to the target document list. This allows later processing steps to correctly identify and transform them as deletion blocks.
<switch:python>
[python] def merge_delete(del_chunks, doc):
    """ Adds the text chunks in del_chunks to the document doc (another
    list of text chunks) with marker to show it is a delete.
    cleanup_delete later resolves these markers into <del> tags."""
    doc.append(DEL_START)
    doc.extend(del_chunks)
    doc.append(DEL_END)

[english] Note that this function modifies the input `doc` list directly in place, so you should ensure you are passing a copy if you need to keep the original state of your document untouched.

[english] If the input is empty, the function will still add both the start and end markers as standalone elements to the doc list. The logic remains safe even without content.
