[english] How can I convert a dictionary of statistical metrics into a formatted list of rows suitable for generating an HTML report table?
[english] This function processes a dictionary where values are stored, pulling only specific keys defined in columns. It formats floating point numbers to three decimal places and then constructs tuples representing table rows.
<switch:python>
[python] def table_lines_from_stats(stats, _, columns):
    """get values listed in <columns> from <stats> and <old_stats>,
    and return a formated list of values, designed to be given to a
    ureport.Table object
    """
    lines = []
    for m_type in columns:
        new = stats[m_type]
        new = "%.3f" % new if isinstance(new, float) else str(new)
        lines += (m_type.replace("_", " "), new, "NC", "NC")
    return lines
