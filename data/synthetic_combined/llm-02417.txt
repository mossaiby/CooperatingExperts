[english] How can I efficiently read records from a large formatted file (like CSV or JSON) and output them as a newline-delimited stream of separate JSON objects without saving everything to disk first?
[english] This function processes an input data file entry by entry, converting each record into a UTF-8 encoded JSON string. It stores these streamed JSON objects within a temporary file handle.
<switch:python>
[python] def transcode_to_stream(input_filename, date_format=None):
    """
    Read a JSON or CSV file and convert it into a JSON stream, which will
    be saved in an anonymous temp file.
    """
    tmp = tempfile.TemporaryFile()
    for entry in open_json_or_csv_somehow(input_filename,
                                          date_format=date_format):
        tmp.write(json.dumps(entry, ensure_ascii=False).encode('utf-8'))
        tmp.write(b'\n')
    tmp.seek(0)
    return tmp
