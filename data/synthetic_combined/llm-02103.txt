[english] How do I ensure a directory structure exists on the filesystem without causing an error if some subdirectory is already present?
[english] To safely create nested directories, we use os.makedirs() to handle recursion. We wrap this call in a try-except block to specifically catch cases where the operating system reports that the path already exists.

[python] def _mkdir_p(path):
    """mkdir -p path"""
    try:
        os.makedirs(path)
    except OSError as exc:
        if exc.errno == errno.EEXIST and os.path.isdir(path):
            pass
        else:
            raise
    else:
        logger.info("New: %s%s", path, os.path.sep)
