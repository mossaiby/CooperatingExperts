[english] How can I reliably check two NumPy arrays to ensure they have identical shapes and their numerical values are close?
[english] To perform this comparison safely, we first explicitly verify that both inputs share exactly the same dimensions using `.shape`. If the geometry matches, we then delegate the proximity testing to a specialized library function.
<switch:python>
[python] def allclose(a, b):
    """
    Test that a and b are close and match in shape.

    Parameters
    ----------
    a : ndarray
        First array to check

    b : ndarray
        First array to check
    """
    from numpy import allclose
    return (a.shape == b.shape) and allclose(a, b)
