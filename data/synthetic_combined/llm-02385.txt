[english] How can I implement a fixed-size cache or sliding window that tracks unique items and reports if an incoming item is new?
[english] This method manages a rolling list (`self.filter`) constrained by `self.max_items`. When processing data, it first checks for existence; if the data is new, it adds it to the end while dropping the oldest element from the front to maintain capacity.

[python] def update(self, data):
        "returns True if unknown"
        if data not in self.filter:
            self.filter.append(data)
            if len(self.filter) > self.max_items:
                self.filter.pop(0)
            return True
        else:
            self.filter.append(self.filter.pop(0))
            return False
