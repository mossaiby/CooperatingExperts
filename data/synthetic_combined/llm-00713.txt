[english] How can I extract all instances of a specific HTML tag, along with its attributes and inner content, from a large HTML string?
[english] This function employs regular expressions to locate matching tags within the input HTML string. It iterates through matches found by `tag_re`, then uses another regex (`attr_re`) to parse attributes for each discovered element.

[python] def itertags(html, tag):
    """
    Brute force regex based HTML tag parser. This is a rough-and-ready searcher to find HTML tags when
    standards compliance is not required. Will find tags that are commented out, or inside script tag etc.

    :param html: HTML page
    :param tag: tag name to find
    :return: generator with Tags
    """
    for match in tag_re.finditer(html):
        if match.group("tag") == tag:
            attrs = dict((a.group("key").lower(), a.group("value")) for a in attr_re.finditer(match.group("attr")))
            yield Tag(match.group("tag"), attrs, match.group("inner"))

[english] What happens if an attribute value itself contains quotes?
[english] Since this uses regex, it relies on specific patterns defined in `attr_re`. If the internal pattern doesn't account for escaped or nested quotes, those values will likely be parsed incorrectly. To handle complex HTML, using a proper library like BeautifulSoup is recommended.
