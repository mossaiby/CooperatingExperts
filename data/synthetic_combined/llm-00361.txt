[english] I need a function that retrieves the current state, exit code, and details for a specific container group identified by its resource group name and instance name.
[english] This function starts by calling `self._get_instance_view` to locate a representation of the target container. Once retrieved, it extracts and returns three attributes—state, exitCode, and detailStatus—packed into a tuple summarizing its status.
<switch:python>
[python] def get_state_exitcode_details(self, resource_group, name):
        """
        Get the state and exitcode of a container group

        :param resource_group: the name of the resource group
        :type resource_group: str
        :param name: the name of the container group
        :type name: str
        :return: A tuple with the state, exitcode, and details.
            If the exitcode is unknown 0 is returned.
        :rtype: tuple(state,exitcode,details)
        """
        current_state = self._get_instance_view(resource_group, name).current_state
        return (current_state.state,
                current_state.exit_code,
                current_state.detail_status)
