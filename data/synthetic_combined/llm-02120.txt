[english] How can I check if a given piece of data ends with any of several predefined line terminator characters?
[english] The function iterates through self.LINE_TERMINATORS, checking each one against the input `data`. It returns the first matching terminal it finds.
<switch:python>
[python] def suffix_line_terminator(self, data):
        """
        Return line terminator data ends with or None.
        """
        for t in self.LINE_TERMINATORS:
            if data.endswith(t):
                return t

        return None

[english] If no terminators match the end of the string, it safely falls back to returning None.
