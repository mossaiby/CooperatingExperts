[english] How can I define diagram generation logic for an entire project within a visitor pattern?
[english] This method acts as a visitor for a Project node, determining which diagrams are relevant depending on the project's structure and configuration.
<switch:python>
[python] def visit_project(self, node):
        """visit a pyreverse.utils.Project node

        create a diagram definition for packages
        """
        mode = self.config.mode
        if len(node.modules) > 1:
            self.pkgdiagram = PackageDiagram("packages %s" % node.name, mode)
        else:
            self.pkgdiagram = None
        self.classdiagram = ClassDiagram("classes %s" % node.name, mode)
