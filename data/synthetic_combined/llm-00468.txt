[english] I need a method that dynamically orders my database query based on user preferences for ascending or descending sort.
[english] This function attempts to modify an existing SQLAlchemy query by checking if specific sorting parameters (`self.asc` and `self._selected`) are defined. It then applies the appropriate ordering using methods like `order_by()` or `desc().order_by()`.
<switch:python>
[python] def items(self):
        """Get query with correct ordering."""
        if self.asc is not None:
            if self._selected and self.asc:
                return self.query.order_by(self._selected)
            elif self._selected and not self.asc:
                return self.query.order_by(desc(self._selected))
        return self.query
