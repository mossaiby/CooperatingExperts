[english] I need to extract and yield all token objects that represent character offsets from a list of sentence structures.
[english] This function employs nested iteration to traverse every token within every provided sentence. It uses conditional yielding to ensure only tokens marked with the specific character offset type are returned.
<switch:python>
[python] def sentences_to_char_tokens(si_sentences):
    '''Convert stream item sentences to character ``Offset``s.'''
    for sentence in si_sentences:
        for token in sentence.tokens:
            if OffsetType.CHARS in token.offsets:
                yield token
