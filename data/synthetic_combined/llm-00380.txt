[english] How can I write a parser function to verify and consume that the next character in my input stream falls within a defined alphabetical range?
[english] This method attempts to match the next available character against the `begin` and `end` boundaries. It first checks for end-of-file safety before peeking at the current stream position (`c`). If `c` is within the inclusive bounds, it consumes the character and signals success.

[python] def read_range(self, begin: str, end: str) -> int:
        """
        Consume head byte if it is >= begin and <= end else return false
        Same as 'a'..'z' in BNF
        """
        if self.read_eof():
            return False
        c = self._stream.peek_char
        if begin <= c <= end:
            self._stream.incpos()
            return True
        return False
