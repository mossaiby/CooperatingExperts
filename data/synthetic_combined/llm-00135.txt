[english] How can I safely add a new info widget while ensuring that its assigned index does not conflict with existing widgets?
[english] This function first retrieves the intended index for the widget. It then enters a loop to search through all current keys in storage until it finds an index that is not already occupied.

[python] def add_info_widget(self, widget):
        '''
        Add widget string to right panel of the screen
        '''
        index = widget.get_index()
        while index in self.info_widgets.keys():
            index += 1
        self.info_widgets[widget.get_index()] = widget

[english] The loop continues incrementing the index until a unique, unused key (index) is found and assigned to the widget within the internal dictionary.
