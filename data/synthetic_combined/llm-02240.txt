[english] The use of `safer_name` and `safer_version` is a good way to prevent invalid characters, like slashes or colons, from appearing in the resulting identifier across different package ecosystems.
[english] What happens if `self.distribution.get_name()` returns an empty string?
[english] If the project name is missing, it will be treated as an empty string during the join operation, so you would receive just the sanitized version number instead.
<switch:python>
[python] def wheel_dist_name(self):
        """Return distribution full name with - replaced with _"""
        components = (safer_name(self.distribution.get_name()), 
                       safer_version(self.distribution.get_version())) 
        # Filter out empty strings before joining to avoid excessive hyphens
        return '-'.join([comp for comp in components if comp]) # Returns only version if name is empty
