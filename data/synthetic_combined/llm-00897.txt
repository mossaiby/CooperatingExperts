[english] How can I normalize a single test expression from my parsing tree into a consistent list of AST statements?
[english] This function ensures that we always get back a list of statements, even if the input is just one piece of code. It checks if the input `pt` is already iterable; otherwise, it wraps the expression inside an 'if not' structure.
<switch:python>
[python] def _clause(self, pt: parsing.ParserTree) -> [ast.stmt]:
        """Normalize a test expression into a statements list.

        Statements list are returned as-is.
        Expression is packaged as:
        if not expr:
            return False
        """
        if isinstance(pt, list):
            return pt
        return [ast.If(ast.UnaryOp(ast.Not(), pt),
                       [self.__exit_scope()],
                       [])]

[english] The function guarantees a statement list output. Remember that the resulting AST block will always be conditionally entered based on the truthiness of the initial expression.
