[english] How can I reliably calculate the absolute end of a given day (23:59:59) using its existing datetime object?
[english] To determine the end of the day, we extract the year, month, and day components from your input timestamp. We then construct a completely new datetime object, manually setting the time to 23 hours, 59 minutes, and 59 seconds.
<switch:python>
[python] def get_end_of_day(timestamp):
    """
    Given a date or a datetime, return a datetime at 23:59:59 on that day
    """
    return datetime.datetime(timestamp.year, timestamp.month, timestamp.day, 23, 59, 59)

[english] This approach is straightforward but requires you to explicitly reconstruct the date and time rather than modifying the original object in place.
