[english] How can an AI actor move through a 3D environment based on its current rotational state and a specific distance?
[english] This method calculates movement vectors by converting degrees into radians, then uses sine and cosine functions to determine directional components for each axis.

        """
        Moves the actor using standard trigonometry along the current rotational vector.
        
        :param float dist: Distance to move

        dy = math.sin(y_angle)
        dy = 0.0
        dx = math.cos(x_angle)
        dz = math.sin(x_angle)
        dx,dy,dz = dx*dist,dy*dist,dz*dist
        x,y,z = self._pos
        self.pos = x+dx,y+dy,z+dz
        return dx,dy,dz
