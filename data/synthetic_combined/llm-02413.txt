[english] How do I safely retrieve a value from a property given its name, ensuring that it returns nothing if the property isn't available?
[english] This function starts by attempting to locate the desired attribute using `self.find_property`. It then checks if this object was found before calling `.get_value()` on it, which prevents errors for missing properties.

[python] def get_property(self, property_name):
        """
        Get a property's value.

        property_name -- the property to get the value of

        Returns the properties value, if found, else None.
        """
        prop = self.find_property(property_name)
        if prop:
            return prop.get_value()

        return None
