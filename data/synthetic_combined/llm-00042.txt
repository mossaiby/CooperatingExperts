[english] How can I automatically generate the required Python import Abstract Syntax Tree (AST) nodes needed to set up module aliases?
[english] This function is a generator that constructs `ast.Import` objects programmatically, allowing you to define how external modules should be imported and aliased in your code.

[python] def _module_imports(ctx: GeneratorContext) -> Iterable[ast.Import]:
    """Generate the Python Import AST node for importing all required
    language support modules."""
    # Yield `import basilisp` so code attempting to call fully qualified
    # `basilisp.lang...` modules don't result in compiler errors
    yield ast.Import(names=[ast.alias(name="basilisp", asname=None)])
    for imp in ctx.imports:
        name = imp.key.name
        alias = _MODULE_ALIASES.get(name, None)
        yield ast.Import(names=[ast.alias(name=name, asname=alias)])
