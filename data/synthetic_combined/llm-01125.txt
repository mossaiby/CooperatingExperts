[english] I need to find all evidence associated with a list of genetic variants, including any suggested changes to phenotypes for those pieces of evidence.
[english] This function retrieves variant information, aggregates all related evidence, and then iteratively fetches potential phenotypic modifications via an API call for each piece of evidence. It compares the current phenotypes against suggested ones to identify additions or deletions.
<switch:python>
[python] def get_variant_phenotypes_with_suggested_changes(variant_id_list):
    '''for each variant, yields evidence and associated phenotypes, both current and suggested'''
    variants = civic.get_variants_by_ids(variant_id_list)
    evidence = list()
    for variant in variants:
        evidence.extend(variant.evidence)
    for e in evidence:
        suggested_changes_url = f'https://civicdb.org/api/evidence_items/{e.id}/suggested_changes'
        resp = requests.get(suggested_changes_url)
        resp.raise_for_status()
        suggested_changes = dict()
        for suggested_change in resp.json():
            pheno_changes = suggested_change['suggested_changes'].get('phenotype_ids', None)
            if pheno_changes is None:
                continue
            a, b = pheno_changes
            added = set(b) - set(a)
            deleted = set(a) - set(b)
            rid = suggested_change['id']
            suggested_changes[rid] = {'added': added, 'deleted': deleted}
        yield e, {'suggested_changes': suggested_changes, 'current': set([x.id for x in e.phenotypes])}

[english] The function yields pairs containing the original evidence item and a dictionary summarizing its current state versus any new suggestions found in its associated changes.
[english] What happens if one of these API calls fails, like receiving an HTTP 404 error?
[english] If `requests.get` returns a failed status, `resp.raise_for_status()` will immediately raise an exception and stop the generator from processing further evidence.
[english] So, it's safest to handle those errors outside this function.
