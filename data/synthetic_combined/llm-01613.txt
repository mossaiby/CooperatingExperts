[english] How can I track an item by name as it is being 'consumed' from a list of pending items?
[english] To mark something as consumed, we add its reference to a specific dictionary and then remove it from the consumption tracker.
<switch:python>
[python] def mark_as_consumed(self, name, new_node):
        """
        Mark the name as consumed and delete it from
        the to_consume dictionary
        """
        self.consumed[name] = new_node
        del self.to_consume[name]
<switch:english>
[english] This function simultaneously updates state by saving data while ensuring the name isn't processed again later through deletion.
[english] What if 'name' is not found in `self.to_consume` when this method is called?
[english] If `name` is missing from `self.to_consume`, calling `del self.to_consume[name]` will result in a KeyError which should be handled using try-except blocks or .pop().
