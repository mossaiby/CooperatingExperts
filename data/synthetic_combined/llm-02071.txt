[english] I need to write a method that calculates the total frame size for my data object, ensuring it accounts for all internal packets and any necessary headers.
[english] This function works by starting with an initial header size of 27. It then iterates over each packet in the instance, calculating how many additional meta-data bits are needed based on whether a packet exceeds 255 bytes.
<switch:python>
[python] def size(self):
        """Total frame size."""

        header_size = 27 # Initial header size
        for datum in self.packets:
            quot, rem = divmod(len(datum), 255)
            header_size += quot + 1
        if not self.complete and rem == 0:
            # Packet contains a multiple of 255 bytes and is not
            # terminated, so we don't have a \x00 at the end.
            header_size -= 1
        header_size += sum(map(len, self.packets))
        return header_size
