[english] How can I calculate an appropriate bin size for continuous feature data using the Freedman-Diaconis rule?
[english] This function implements the FD method to determine the optimal bin width based on the Interquartile Range (IQR) and sample size.
It first calculates Q75 and Q25 from your input values using numpy's percentile calculation.
<switch:python>
[python] def FreedmanDiaconisBinSize(feature_values):
  """
  The bin size in FD-binning is given by size = 2 * IQR(x) * n^(-1/3)
  More Info: https://en.wikipedia.org/wiki/Freedman%E2%80%93Diaconis_rule

  If the BinSize ends up being 0 (in the case that all values are the same),
  return a BinSize of 1.
  """

  q75, q25 = numpy.percentile(feature_values, [75, 25])
  IQR = q75 - q25

  return 2.0 * IQR * len(feature_values) ** (-1.0/3.0)
