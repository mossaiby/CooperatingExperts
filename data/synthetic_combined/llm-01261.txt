[english] How can I calculate accuracy when my model outputs continuous probability values instead of discrete categories?
[english] To determine if your predictions match reality, this function first rounds every value in y_predicted to the nearest integer. Then it compares these rounded predictions against the true labels element by element.

[python] def round_accuracy(y_true, y_predicted):
    """
    Rounds predictions and calculates accuracy in terms of absolute coincidence.

    Args:
        y_true: list of true values
        y_predicted: list of predicted values

    Returns:
        portion of absolutely coincidental samples
    """
    predictions = [round(x) for x in y_predicted]
    examples_len = len(y_true)
    correct = sum([y1 == y2 for y1, y2 in zip(y_true, predictions)])
    return correct / examples_len if examples_len else 0
