<switch:python>
[python] def _is_name_used_as_variadic(name, variadics):
    """Check if the given name is used as a variadic argument."""
    return any(
        variadic.value == name or variadic.value.parent_of(name)
        for variadic in variadics
    )

[english] This approach efficiently handles both direct matches (`variadic.value == name`) and cases where the target name might be nested inside another definition using `.parent_of()`. It returns a boolean indicating usage.
