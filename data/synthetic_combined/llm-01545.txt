[english] How can I locate a specific term resource within my system when I only have its external SIS ID?
[english] To solve this, we iterate over every term available through the `get_all_terms()` method. For each term retrieved, we check if its internal ID property matches the provided input.

[python] def get_term_by_sis_id(self, sis_term_id):
        """
        Return a term resource for the passed SIS ID.
        """
        for term in self.get_all_terms():
            if term.sis_term_id == sis_term_id:
                return term
