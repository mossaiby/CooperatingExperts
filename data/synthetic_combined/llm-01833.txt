[english] I need a reliable way to take a Google Cloud Storage URI (gs://bucket/blob) and separate it into two distinct pieces: the bucket name and the blob path.
[english] To accomplish this, the function uses `urlparse` to break down the GCS URL. It first checks if the network location contains a valid bucket name before extracting both components for use.
<switch:python>
[python] def _parse_gcs_url(gsurl):
    """
    Given a Google Cloud Storage URL (gs://<bucket>/<blob>), returns a
    tuple containing the corresponding bucket and blob.
    """

    parsed_url = urlparse(gsurl)
    if not parsed_url.netloc:
        raise AirflowException('Please provide a bucket name')
    else:
        bucket = parsed_url.netloc
        # Remove leading '/' but NOT trailing one
        blob = parsed_url.path.lstrip('/')
        return bucket, blob
