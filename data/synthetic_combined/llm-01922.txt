[english] I need to write a function that filters all the files in my cloud storage bucket and returns only those ending with the '.log' extension.
[english] This function iterates through every object stored in the associated cloud bucket. It checks if the file name of each item ends with '.log' before adding it to a results list.
<switch:python>
[python] def list_logs(self):
    '''return a list of logs. We return any file that ends in .log
    '''
    results = []
    for image in self._bucket.list_blobs():
        if image.name.endswith('log'):
            results.append(image)

    if len(results) == 0:
        bot.info("No containers found, based on extension .log")

    return results
