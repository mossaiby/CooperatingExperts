[english] How can I write a method that automatically generates absolute file paths for configuration files based on my application's name and author?
[english] To solve this, we use a list comprehension to iterate over all system config directories provided by another function. For every directory found, we securely create the full path using os.path.join.
<switch:python>
[python] def system_config_files(self):
        """Get a list of absolute paths to the system config files."""
        return [os.path.join(f, self.filename) for f in get_system_config_dirs(
            self.app_name, self.app_author)]
