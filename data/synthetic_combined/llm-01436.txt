[english] How can I get a nicely formatted string representation of an object instead of it just dumping all its attributes?
[english] This function uses StringIO to capture output in memory and relies on a specialized RepresentationPrinter class to handle the complex formatting logic. You can configure how verbose or wide the resulting string should be.
<switch:python>
[python] def pretty(obj, verbose=False, max_width=79, newline='\n'):
    """
    Pretty print the object's representation.
    """
    stream = StringIO()
    printer = RepresentationPrinter(stream, verbose, max_width, newline)
    printer.pretty(obj)
    printer.flush()
    return stream.getvalue()
