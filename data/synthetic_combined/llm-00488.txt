[english] How can I retrieve a specific chat's internal ID just knowing its username?
[english] This function determines the user's internal chat ID by first accessing an external API endpoint via a GET request. It cleans the provided username and then iterates through all results to find a match.
<switch:python>
[python] def get_chat_id(self, username):
        """Lookup chat_id of username if chat_id is unknown via API call."""
        if username is not None:
            chats = requests.get(self.base_url + "/getUpdates").json()
            user = username.split("@")[-1]
            for chat in chats["result"]:
                if chat["message"]["from"]["username"] == user:
                    return chat["message"]["from"]["id"]
