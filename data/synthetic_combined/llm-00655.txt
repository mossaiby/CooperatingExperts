[english] How can I find the type of a specific attribute, ignoring case?
[english] This method handles the retrieval by first standardizing the input to uppercase to ensure case-insensitivity during the search.
<switch:python>
[python] def attribute_type(self, attribute_name):
        '''
        Obtain the type of an attribute.
        '''
        attribute_name = attribute_name.upper()
        for name, ty in self.attributes:
            if name.upper() == attribute_name:
                return ty
