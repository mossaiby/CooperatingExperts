[english] How can I make my camera rotate smoothly based on mouse movement within a fullscreen application?
[english] This method updates the object's rotation by calculating new coordinates based on mouse deltas and configured sensitivity. It ensures that the vertical angle remains clamped while allowing the horizontal angle to wrap around 360 degrees.
<switch:python>
[python] def on_mouse_motion(self, x, y, dx, dy):
        """
        Handles mouse motion and rotates the attached camera accordingly.
        
        For more information about how to customize mouse movement, see the class documentation here :py:class:`WorldViewMouseRotatable()`\ .
        """
        if not self.world.peng.window.exclusive:
            return
        m = self.world.peng.cfg["controls.mouse.sensitivity"]
        x, y = self.rot
        x, y = x + dx * m, y + dy * m
        y = max(-90, min(90, y))
        x %= 360
        newrot = (x,y)
        self.rot= newrot
<switch:english>
[english] The implementation is effective for basic camera control, but note that it relies on pre-existing attributes like self.rot which must be defined externally.
[english] What happens if `self.world.peng.cfg` or keys within it are missing?
[english] If the configuration keys are not present in `self.world.peng.cfg`, a KeyError will be raised when attempting to access 'controls.mouse.sensitivity'. It is safer to use dict.get() methods when retrieving configuration values.
