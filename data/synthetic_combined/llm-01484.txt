[english] How do I generate evenly spaced coordinates for chroma bins given a total count 'n' and specific resolution settings?
[english] We can use numpy's linspace function here, which is perfect for uniformly dividing a range into a set number of points.

[python] def __coord_chroma(n, bins_per_octave=12, **_kwargs):
    '''Get chroma bin numbers'''
    return np.linspace(0, (12.0 * n) / bins_per_octave, num=n+1, endpoint=True)
