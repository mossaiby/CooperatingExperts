[python] def _prep_params(params):
        '''Remove empty (None) valued keywords and self from function parameters'''

        return {k: v for (k, v) in params.items() if v is not None and k != 'self'}

[english] This method efficiently removes both self-references and null values, ensuring only valid data points remain in the resulting parameter set.
