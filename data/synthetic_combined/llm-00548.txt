[english] How do I scan my database to count all coding intervals per chromosome and find the longest one?
[english] This function inspect, counts, and analyzes genomic interval data stored in the adapter. It starts by setting up trackers for total counts and maximum length.
<switch:python>
[python] def intervals(context, build):
    """Show all indexes in the database"""
    LOG.info("Running scout view index")
    adapter = context.obj['adapter']

    intervals = adapter.get_coding_intervals(build)
    nr_intervals = 0
    longest = 0
    for chrom in CHROMOSOMES:
        for iv in intervals[chrom]:
            iv_len = iv.end - iv.begin
            if iv_len > longest:
                longest = iv_len
        int_nr = len(intervals.get(chrom, []))
        click.echo("{0}\t{1}".format(chrom, int_nr))
        nr_intervals += int_nr

    LOG.info("Total nr intervals:%s", nr_intervals)
    LOG.info("Total nr genes:%s", adapter.all_genes(build).count())
    LOG.info("Longest interval:%s", longest)

[english] What happens if a specific chromosome listed in CHROMOSOMES does not have any entry in intervals?
[english] It uses `intervals.get(chrom, [])`, which safely returns an empty list instead of raising a KeyError if no data exists for that chromosome.
