[english] How can I safely retrieve Athena query results once a given execution ID has been submitted?
[english] This function first verifies if the specified query ID is in a valid state using `check_query_status`. It then conditionally proceeds to fetch results only if the execution hasn't failed or is still pending.
<switch:python>
[python] def get_query_results(self, query_execution_id):
        """
        Fetch submitted athena query results. returns none if query is in intermediate state or
        failed/cancelled state else dict of query output

        :param query_execution_id: Id of submitted athena query
        :type query_execution_id: str
        :return: dict
        """
        query_state = self.check_query_status(query_execution_id)
        if query_state is None:
            self.log.error('Invalid Query state')
            return None
        elif query_state in self.INTERMEDIATE_STATES or query_state in self.FAILURE_STATES:
            self.log.error('Query is in {state} state. Cannot fetch results'.format(state=query_state))
            return None
        return self.conn.get_query_results(QueryExecutionId=query_execution_id)
