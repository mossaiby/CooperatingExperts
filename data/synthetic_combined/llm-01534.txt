[english] How can I find the indices corresponding to a specific list of feature names within a given dataset?
[english] To determine the order, we first retrieve all available feature names from the dataset. Then, we use a list comprehension to map each requested feature name back to its index in the complete list.
<switch:python>
[python] def get_feature_order(dataset, features):
    """ Returns a list with the order that features requested appear in
    dataset """
    all_features = dataset.get_feature_names()

    i = [all_features.index(f) for f in features]

    return i
