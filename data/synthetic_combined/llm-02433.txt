[english] How do I display an image from a file so that it persists until I press a key?
[english] This function simplifies image visualization using OpenCV. It combines loading and displaying functionality into one reusable helper.
<switch:python>
[python] def imshow(img, win_name='', wait_time=0):
    """Show an image.

    Args:
        img (str or ndarray): The image to be displayed.
        win_name (str): The window name.
        wait_time (int): Value of waitKey param.
    """
    cv2.imshow(win_name, imread(img))
    cv2.waitKey(wait_time)
<switch:english>
[english] The `cv2.imshow` function opens a window with the image, and `cv2.waitKey(wait_time)` handles controlling how long that display remains active.
[english] What causes OpenCV to crash if the input string is not found?
[english] If `imread(img)` returns None because it failed to find or decode the file, trying to pass None into `cv2.imshow` will usually cause an error.
