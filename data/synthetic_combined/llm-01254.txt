[english] How can I write a script to rename an existing snapshot within an application?
[english] The function first retrieves the snapshot using the old name and verifies it exists; then, it checks if a snapshot with the desired new name is already present before performing the renaming operation.
<switch:python>
[python] def rename(old_name, new_name):
    """Renames a snapshot"""
    app = get_app()

    snapshot = app.get_snapshot(old_name)
    if not snapshot:
        click.echo("Couldn't find snapshot %s" % old_name)
        sys.exit(1)

    new_snapshot = app.get_snapshot(new_name)
    if new_snapshot:
        click.echo("Snapshot with name %s already exists" % new_name)
        sys.exit(1)

    app.rename_snapshot(snapshot, new_name)
    click.echo("Renamed snapshot %s to %s" % (old_name, new_name))
