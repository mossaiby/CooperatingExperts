[english] How can I programmatically add a Bind node to an existing sequence parser tree within my application?
[english] This approach prepends a new 'Bind' node to the sequence's current parser_tree by first resolving a value from the context and then assigning it during the creation of the Bind object.
<switch:python>
[python] def add_bind(self, sequence, cpt):
    """Create a tree.Bind"""
    cpt_value = self.value(cpt)
    sequence.parser_tree = parsing.Bind(cpt_value, sequence.parser_tree)
    return True
<switch:english>
[english] The function returns True upon successful modification, indicating that the binding structure has been successfully integrated into the sequence tree.
[english] What happens if self.value(cpt) raises an error?
[english] If `self.value` fails, the entire process will raise an exception before modifying the parser_tree, and thus the function would fail naturally rather than adding a partial Bind node.
