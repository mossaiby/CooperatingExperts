[python] def is_type(self, value):
    """Returns True if node_type == value.

    If value is a tuple, node_type is checked against each member and True is returned if any of
    them match.
    """
    if isinstance(value, tuple):
      for opt in value:
        if self.node_type == opt:
          return True
      return False
    else:
      return self.node_type == value

[english] The logic ensures that if `value` is a tuple, it iterates through every option to find an exact match for `self.node_type`. If not a tuple, it performs a simple direct comparison.
