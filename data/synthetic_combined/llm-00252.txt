[english] How can I programmatically force a known failed member of a cluster to be marked as having left?
[english] This method interacts with the underlying Nomad API to trigger a forced departure for a specified `node`. It sets up the necessary parameters before executing the POST request.
<switch:python>
[python] def force_leave(self, node):
        """Force a failed gossip member into the left state.

            https://www.nomadproject.io/docs/http/agent-force-leave.html

            returns: 200 status code
            raises:
              - nomad.api.exceptions.BaseNomadException
              - nomad.api.exceptions.URLNotFoundNomadException
        """
        params = {"node": node}
        return self.request("force-leave", params=params, method="post").status_code
