[english] How can I write a helper function that determines if an attribute type in PKCS#11 represents a raw byte array instead of another data format?
[english] To check for binary types, we simply need to exclude all other known simple data formats (like booleans, strings, and numbers).
<switch:python>
[python] def isBin(self, type):
        """
        is the type a byte array value?

        :param type: PKCS#11 type like `CKA_MODULUS`
        :rtype: bool
        """
        return (not self.isBool(type)) \
            and (not self.isString(type)) \
            and (not self.isNum(type))

[english] By checking that the type is not classified as a boolean, string, or number, this function correctly assumes all remaining supported types are binary.
