[english] How do I reverse the normalization process to get the original input values back from standardized data?
[english] To reconstruct the original value, we apply an inverse transformation. We multiply the normalized input by its scale and then offset it using the location.

[python] def _inv_z(self, z):
    """Reconstruct input `x` from a its normalized version."""
    with tf.name_scope("reconstruct"):
      return z * self.scale + self.loc

[english] This function effectively reverses standardization or denormalization, ensuring `z` is mapped back to its expected range relative to `self.loc` and `self.scale`. The usage of `tf.name_scope` helps organize the computation graph in TensorFlow.
[english] What happens if the original data was highly skewed, like using a log transformation instead?
[english] If non-linear transformations were used during normalization, this simple linear inverse won't suffice; you would need to apply the specific inverse of that function within `_inv_z`.
