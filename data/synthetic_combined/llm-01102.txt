[english] How can I run a pre-defined list of shell commands sequentially within a specific directory?
[english] This function iterates through your stored `self.actions` and executes each one as a separate process using Python's `subprocess` module.
<switch:python>
[python] def execute_actions(self, cwd):
        """Iterates over the actions and executes them in order."""
        self._execute_globals(cwd)
        for action in self.actions:
            logger.info("executing {}".format(action))
            p = subprocess.Popen(action, shell=True, cwd=cwd)
            p.wait()

[english] The use of p.wait() ensures that subsequent actions do not start until the previous command has completely finished, which is crucial for sequential task runners.
