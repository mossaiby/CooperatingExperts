[english] How can I reliably retrieve the associated clock signal for this component, even if it’s defined by an ancestor?
[english] This method prioritizes finding a local clock first to avoid unnecessary lookups. If the clock isn't found locally, it attempts a recursive search up the unit hierarchy.

[python] def _getAssociatedClk(self):
        """
        If interface has associated clk return it otherwise
        try to find clk on parent recursively
        """
        a = self._associatedClk

        if a is not None:
            return a

        p = self._parent
        assert p is not None
