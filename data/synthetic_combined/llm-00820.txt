[english] I need to convert a Python object into a string formatted as 'module:qualname' so I can uniquely identify it, even if it is nested within a class.
[english] This utility function aims to provide a canonical import path for objects by first ensuring we focus on the class or method level. It then retrieves the module and combines its name with the object's qualified name using '__qualname__', which handles nesting efficiently.
<switch:python>
[python] def name(obj) -> str:
	"""This helper function attempts to resolve the dot-colon import path for a given object.
	
	Specifically searches for classes and methods, it should be able to find nearly anything at either the module
	level or nested one level deep.  Uses ``__qualname__`` if available.
	"""
	
	if not isroutine(obj) and not hasattr(obj, '__name__') and hasattr(obj, '__class__'):
		obj = obj.__class__
	
	module = getmodule(obj)
	
	return module.__name__ + ':' + obj.__qualname__
