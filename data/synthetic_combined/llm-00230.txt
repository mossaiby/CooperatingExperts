[english] I need a function that calculates the total number of elements in a tensor, or alternatively, the product of sizes along specific dimensions.
[english] This function determines the size of a tensor. If an axis is provided, it finds the size by multiplying the lengths across those specified dimensions using TensorFlow operations.
<switch:python>
[python] def _axis_size(x, axis=None):
  """Get number of elements of `x` in `axis`, as type `x.dtype`."""
  if axis is None:
    return tf.cast(tf.size(input=x), x.dtype)
  return tf.cast(
      tf.reduce_prod(input_tensor=tf.gather(tf.shape(input=x), axis)), x.dtype)

[english] The result is always cast to match the input tensor's data type (`x.dtype`). This allows for consistent usage when mixing shape calculations and other numeric tensors.
[english] What happens if I pass a list of axes, like (0, 1), instead of just one axis?
[english] It calculates the product of the sizes of all dimensions listed in that tuple. For example, passing (0, 1) multiplies the size along dimension 0 by the size along dimension 1.
