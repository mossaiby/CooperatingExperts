[english] How can I reliably check if two files are identical using system commands in Python?
[english] This function uses the external `diff` utility to perform a quick comparison between two file paths. It executes the `diff -q` command and checks the exit code to determine equality.
<switch:python>
[python] def equal(self, cwd):
        """ Returns True if left and right are equal
        """
        cmd = ["diff"]
        cmd.append("-q")
        cmd.append(self.left.get_name())
        cmd.append(self.right.get_name())

        try:
            Process(cmd).run(cwd=cwd, suppress_output=True)
        except SubprocessError as e:
            if e.get_returncode() == 1:
                return False
            else:
                raise e
        return True
<switch:english>
[english] The logic relies on the subprocess failing (return code 1) only when content differences are found, otherwise it assumes they match.
[english] What happens if one of the files specified by self.left or self.right does not exist?
[english] If a file is missing, `diff` will likely throw a 'file not found' error before returning a standard exit code. You should wrap it in broader exception handling to catch such OS-level errors.
