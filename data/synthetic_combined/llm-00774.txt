[english] How do I programmatically find parent classes in a method resolution order that meet specific configuration criteria?
[english] This function iterates through all ancestors of an input class using its Method Resolution Order (MRO). For each ancestor, it checks if the class is derived from SingletonConfigurable while making sure we exclude the base configuration type itself.

[python] def _walk_mro(cls):
        """Walk the cls.mro() for parent classes that are also singletons

        For use in instance()
        """

        for subclass in cls.mro():
            if issubclass(cls, subclass) and \
                    issubclass(subclass, SingletonConfigurable) and \
                    subclass != SingletonConfigurable:
                yield subclass
