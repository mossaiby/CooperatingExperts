[python] def error_parsing(msg="unknown options"):
    """
    Print any parsing error and exit with status -1
    """
    sys.stderr.write("Error parsing command line: %s\ntry 'mongotail --help' for more information\n" % msg)
    sys.stderr.flush()
    exit(EINVAL)
<switch:english>
[english] The use of `sys.stderr` ensures that errors do not mix with successful program output, while exiting with an error code like `EINVAL` signals failure to calling processes.
