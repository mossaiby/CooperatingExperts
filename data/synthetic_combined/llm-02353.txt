[english] How do I temporarily inject extra state or 'context' into a processing block without manually managing the cleanup?
[english] This function handles temporary context by first pushing the provided `additional` data using internal `push` logic. It then yields the current state (`self.current`) and ensures the added context is removed afterwards.
<switch:python>
[python] def additional(self, additional, use_parent=False):
        """
        Allows temporarily pushing an additional context, yields the new context
        into the following block.
        """
        self.push(additional, use_parent)
        yield self.current
        self.pop()
