[english] How can I explicitly add a specific directory within my Python build artifacts, like ./build/lib.<platform>, to be discoverable by the interpreter?
[english] This function modifies Python's search path (`sys.path`) to include a build-specific library folder. It dynamically constructs this path using information about your system and current environment.

[python] def addbuilddir():
    """Append ./build/lib.<platform> in case we're running in the build dir
    (especially for Guido :-)"""
    from distutils.util import get_platform
    s = "build/lib.%s-%.3s" % (get_platform(), sys.version)
    if hasattr(sys, 'gettotalrefcount'):
        s += '-pydebug'
    s = os.path.join(os.path.dirname(sys.path[-1]), s)
    sys.path.append(s)
