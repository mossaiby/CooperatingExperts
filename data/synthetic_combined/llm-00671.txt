[python] def start_instances(instances, region):
    '''Start all the instances given by its ids'''
    if not instances: return
    conn = ec2_connect(region)
    log("Starting instances {0}.".format(instances))
    conn.start_instances(instances)
    log("Done")

[english] The function uses the `ec2_connect` helper to establish a connection and then calls `conn.start_instances(instances)` directly to trigger the start command on AWS.
