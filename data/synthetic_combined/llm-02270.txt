[english] This function first fetches configuration details from internal state, such as server hosts and credentials. It then utilizes those retrieved attributes to create and return a live database connection.

[python] def get_conn(self):
        """
        Returns a mssql connection object
        """
        conn = self.get_connection(self.mssql_conn_id)
        conn = pymssql.connect(
            server=conn.host,
            user=conn.login,
            password=conn.password,
            database=self.schema or conn.schema,
            port=conn.port)
        return conn

[english] The method assumes that the call to self.get_connection() will successfully retrieve a dictionary containing all necessary connection parameters.
