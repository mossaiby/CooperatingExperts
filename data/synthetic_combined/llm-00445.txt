[english] How can I register a custom freeform key to correctly handle complex MP4 Tags metadata mapping?
[english] This function helps standardize how specific, non-traditional MP4 keys (freeform atoms) are accessed within your tag structure by defining explicit getter and setter functions. It calculates its unique atom ID based on the user provided name, mean, and internal logic before registering it with the class.
<switch:python>
[python] def RegisterFreeformKey(cls, key, name, mean=b"com.apple.iTunes"):
        """Register a text key.

        If the key you need to register is a simple one-to-one mapping
        of MP4 freeform atom (----) and name to EasyMP4Tags key, then
        you can use this function::

            EasyMP4Tags.RegisterFreeformKey(
                "musicbrainz_artistid", b"MusicBrainz Artist Id")
        """
        atomid = b"----:" + mean + b":" + name

        def getter(tags, key):
            return [s.decode("utf-8", "replace") for s in tags[atomid]]

        def setter(tags, key, value):
            tags[atomid] = [utf8(v) for v in value]

        def deleter(tags, key):
            del(tags[atomid])

        cls.RegisterKey(key, getter, setter, deleter)
