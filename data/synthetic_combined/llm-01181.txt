[english] How can I take a list of lists or tuples and output them as tab-separated values to standard output or a file?
[english] To achieve TSV formatting, we iterate over every record in the input table. Inside the loop, we use the built-in print function with the unpacking operator (*) and specify the custom separator.
<switch:python>
[python] def printtsv(table, sep="\t", file=sys.stdout):
    """stupidly print an iterable of iterables in TSV format"""
    for record in table:
        print(*record, sep=sep, file=file)
<switch:english>
[english] This approach is concise for printing structured data to various streams, though non-string elements will be automatically converted to strings by Python's print mechanism.
[english] If one of the records in the input table is an empty list, what exact output will that line produce?
[english] An empty record like `[]` will result in a single newline character being printed because no positional arguments are passed to the unpack operator, satisfying print's default behavior.
