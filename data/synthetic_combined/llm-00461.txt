[python] def conjugate(self):
        """Return the conjugate of the QuantumChannel."""
        return Choi(np.conj(self._data), self.input_dims(), self.output_dims())

[english] This method generates a completely new `Choi` instance based on the conjugated state of the channel's core matrix, which is useful for certain quantum operations.
[english] If the internal data (`self._data`) represents a purely real matrix, does using np.conj() actually change the resulting conjugate?
[english] No, complex conjugation applied to a purely real number returns that same value. Therefore, if your channel is defined entirely by real parameters, the conjugate will be identical to the original channel.
