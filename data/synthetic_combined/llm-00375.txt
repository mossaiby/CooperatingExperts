[english] How can I write a function that safely modifies an existing domain object based on its name while optionally changing its salt or username?
[english] I implemented this by first fetching the domain from the database using its name to ensure it exists. Then, I conditionally apply modifications for new salts and usernames.
<switch:python>
[python] def modify_domain(self, domain_name, new_salt=False, username=None):
        """ Modify an existing domain.

        :param domain_name: The name of the domain to modify.
        :param new_salt: Whether to generate a new salt for the domain.
        :param username: If given, change domain username to this value.
        :returns: The modified :class:`Domain <pwm.core.Domain>` object.
        """
        domain = self._get_domain_from_db(domain_name)
        if domain is None:
            raise NoSuchDomainException
        if new_salt:
            _logger.info("Generating new salt..")
            domain.new_salt()
        if username is not None:
            domain.username = username
        return domain
