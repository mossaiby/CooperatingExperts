[english] How do I reliably get a mapping of entry point names to their corresponding objects from all members of a specific group?
[english] This function first fetches every available entry point using an assumed helper, `get_group_all`. It then systematically builds a dictionary, checking for existing keys before adding any new ones.

[python] def get_group_named(group, path=None):
    """Find a group of entry points with unique names.

    Returns a dictionary of names to :class:`EntryPoint` objects.
    """
    result = {}
    for ep in get_group_all(group, path=path):
        if ep.name not in result:
            result[ep.name] = ep
    return result

[english] The explicit check 'if ep.name not in result:' ensures that if the source list contains duplicate names, only the first instance encountered is stored.
