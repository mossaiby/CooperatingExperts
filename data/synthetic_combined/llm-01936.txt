[english] How can I iterate through multiple source directories to generate corresponding output files?
[english] This approach iterates over paired input and output directories, using `zip` to ensure that each set of paths generated by `os.walk` is processed against its correct target directory.

[python] def process_directories(self):
        """Create the rst files from the input directories in the
        :attr:`in_dir` attribute"""
        for i, (base_dir, target_dir, paths) in enumerate(zip(
                self.in_dir, self.out_dir, map(os.walk, self.in_dir))):
            self._in_dir_count = i
            self.recursive_processing(base_dir, target_dir, paths)
