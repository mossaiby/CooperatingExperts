[english] How do I set up a command-line tool that requires users to provide an RSS feed URL?
[english] This helper function initializes and configures an argument parser tailored for backend commands, specifically requiring one mandatory positional argument.
<switch:python>
[python] def setup_cmd_parser(cls):
        """Returns the RSS argument parser."""

        parser = BackendCommandArgumentParser(cls.BACKEND.CATEGORIES,
                                              archive=True)

        # Required arguments
        parser.parser.add_argument('url',
                                   help="URL of the RSS feed")

        return parser
