[english] How do I convert my collection of dictionaries into a clean, viewable HTML table within a Jupyter Notebook?
[english] This function is designed to generate an HTML table representation of the object's internal state. It first collects all unique column headers from its data, then iterates through each record to construct individual tr elements.
<switch:python>
[python] def _repr_html_(self):
        """
        Jupyter Notebook magic repr function.
        """
        all_keys = list(set(itertools.chain(*[d.keys for d in self])))
        rows = ''
        for decor in self:
            th, tr = decor._repr_html_row_(keys=all_keys)
            rows += '<tr>{}</tr>'.format(tr)
        header = '<tr>{}</tr>'.format(th)
        html = '<table>{}{}</table>'.format(header, rows)
        return html
