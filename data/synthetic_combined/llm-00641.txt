[english] To detect duplicates, we create a set to track keys encountered so far. We then iterate through the dictionary items, extracting the actual value of constant keys before comparing them against our tracking set.

[python] def visit_dict(self, node):
        """check duplicate key in dictionary"""
        keys = set()
        for k, _ in node.items:
            if isinstance(k, astroid.Const):
                key = k.value
                if key in keys:
                    self.add_message("duplicate-key", node=node, args=key)
                keys.add(key)
