[english] How can I extract a list of starting timestamps from a complex JSON file structure into a numerical array?
[english] This function reads the specified JSON file, loads its contents, and then iteratively accesses the 'start' key for every object within the 'beats' list. Finally, it converts this extracted list into a NumPy array.
<switch:python>
[python] def json_to_beats(beats_json_file):
    """Extracts the beats from the beats_json_file and puts them into
        an np array."""
    f = open(beats_json_file, "r")
    beats_json = json.load(f)
    beats = []
    for beat in beats_json["beats"]:
        beats.append(beat["start"])
    f.close()
    return np.asarray(beats)
