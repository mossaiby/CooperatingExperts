[python] def update_settings(self, integration_id, settings):
        """Updates settings for given integration."""

        (integration, created) = self.get_or_create(integration_id=integration_id)

        try:
            current_settings = json.loads(integration.settings)
        except ValueError:
            current_settings = {}

        current_settings.update(settings)

        integration.settings = json.dumps(current_settings)

        integration.save()

[english] The function then overwrites the integration's settings with the updated JSON string and saves it back to the database.
[english] What happens if we pass a setting that is not serializable by JSON?
[english] Python's `json.dumps()` will raise a TypeError if any value in `settings` cannot be converted (like objects or functions), which you would need an error handler for.
