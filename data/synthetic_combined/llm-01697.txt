[english] I need a function that determines if a given (key, value) pair duplicates an existing item when performing deduplication.
[english] This utility function checks for duplication by comparing internal pointers within the `dedup_result` structure. It specifically uses pointer equality between node flags to confirm if an existing match has been found.
<switch:python>
[python] def _isdupitem(self, key, val, dedup_result):
        """Return whether (key, val) duplicates an existing item."""
        isdupkey, isdupval, nodeinv, nodefwd = dedup_result
        isdupitem = nodeinv is nodefwd
        if isdupitem:
            assert isdupkey
            assert isdupval
        return isdupitem
