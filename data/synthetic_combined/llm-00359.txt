[english] How can I inspect and retrieve the actual content of a specific key in my database without knowing its data structure beforehand?
[english] The function first uses `DB.type(key)` to determine if the stored data is meant to be a collection (`set`) or a single value (`string`). It then conditionally calls either `DB.smembers` or `DB.get` and prints the result using color formatting.
<switch:python>
[python] def do_DBKEY(self, key):
        """Print raw content of a DB key.
        DBKEY g|u09tyzfe"""
        type_ = DB.type(key).decode()
        if type_ == 'set':
            out = DB.smembers(key)
        elif type_ == 'string':
            out = DB.get(key)
        else:

        print('value:', white(out))
