[english] To compare a batch of incoming images against our stored features, we perform matrix multiplication using the NumPy library. We treat both input sets as matrices and compute their relationships through the dot product.

[python] def _dot_product(self, imgs_to_decode):
        """ Decoding using the dot product.
        """
        return np.dot(imgs_to_decode.T, self.feature_images).T

[english] By transposing both the input and stored features before the multiplication, this function calculates a similarity score between every new image and every existing feature.
[english] What happens if `imgs_to_decode` is an empty list or array?
[english] If `imgs_to_decode` is empty, the `.T` operation will result in an empty matrix, and the resulting dot product calculation will yield a zero-dimensional or empty output depending on the shape of `self.feature_images`.
