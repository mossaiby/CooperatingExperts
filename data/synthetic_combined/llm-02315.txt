[english] How can I write a Python script to reboot multiple EC2 instances within a specific AWS region?
[english] This function automates resource maintenance by targeting a list of instance IDs in a specified AWS location. It uses helper functions to establish an connection and then executes a bulk reboot command on the provided instance list while tracking progress via logging.
<switch:python>
[python] def reboot_instances(instances, region):
    '''Reboot all the instances given by its ids'''
    if not instances: return
    conn = ec2_connect(region)
    log("Rebooting instances {0}.".format(instances))
    conn.reboot_instances(instances)
    log("Done")
