[english] How can I count the number of elements within a structure that belong to a specific type named 'Type'?
[english] To solve this, we iterate through all values in `self._hsig`. We then check if each item’s class name is exactly 'Type', keeping a running total.
<switch:python>
[python] def count_types(self) -> int:
        """ Count subtypes """
        n = 0
        for s in self._hsig.values():
            if type(s).__name__ == 'Type':
                n += 1
        return n
