[english] The function assumes a global dictionary `CYTOBANDS` exists and populates the result by iterating over intervals found at that position. This means it only returns the data from the last (or only) interval processed.
[english] What would happen if multiple cytobands overlap at the same position?
[english] Currently, the function will overwrite the `coordinate` variable with each subsequent interval in the loop.
<switch:python>
[python] def get_cytoband_coordinates(chrom, pos):
    # Return a list of all overlapping coordinates instead of the last one
    return [interval.data for interval in CYTOBANDS[chrom][pos]] if chrom in CYTOBANDS and pos in CYTOBANDS[chrom] else []
