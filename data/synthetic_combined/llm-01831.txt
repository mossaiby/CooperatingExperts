[english] How can I extract structured reference data, including player positions and IDs, from the 'officials' table on the current game page?
[english] To get this information, we first retrieve the full document using `self.get_doc()` and then isolate the specific HTML table labeled with ID 'officials'. This raw table is passed to a utility for parsing.
<switch:python>
[python] def ref_info(self):
        """Gets a dictionary of ref positions and the ref IDs of the refs for
        that game.

        :returns: A dictionary of ref positions and IDs.
        """
        doc = self.get_doc()
        table = doc('table#officials')
        return sportsref.utils.parse_info_table(table)
