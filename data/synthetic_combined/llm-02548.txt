[english] How do I map a continuous range of floating point numbers into a specific number of discrete integer levels?
[english] This function quantizes an array by first clipping values to a defined range and then calculating their percentage within that range. It scales this to the desired levels before rounding down and ensuring the result does not exceed the maximum level.
<switch:python>
[python] def quantize(arr, min_val, max_val, levels, dtype=np.int64):
    """Quantize an array of (-inf, inf) to [0, levels-1].

    Args:
        arr (ndarray): Input array.
        min_val (scalar): Minimum value to be clipped.
        max_val (scalar): Maximum value to be clipped.
        levels (int): Quantization levels.
        dtype (np.type): The type of the quantized array.

    Returns:
        tuple: Quantized array.
    """
    if not (isinstance(levels, int) and levels > 1):
        raise ValueError(
            'levels must be a positive integer, but got {}'.format(levels))
    if min_val >= max_val:
        raise ValueError(
            'min_val ({}) must be smaller than max_val ({})'.format(
                min_val, max_val))

    arr = np.clip(arr, min_val, max_val) - min_val
    quantized_arr = np.minimum(
        np.floor(levels * arr / (max_val - min_val)).astype(dtype), levels - 1)

    return quantized_arr

[english] The use of `np.clip` ensures inputs are contained, while the final application of `np.minimum(..., levels - 1)` guarantees no value exceeds the upper boundaries of your quantization scale.
