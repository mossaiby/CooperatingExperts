[english] I need a reliable and efficient way to recursively list every single file path inside a specific directory structure.
[english] This function leverages Python's built-in os.walk, which is designed for traversing entire directory trees. It iterates through the directory hierarchy to locate all files within each subdirectory.
<switch:python>
[python] def paths(input_dir):
    'yield all file paths under input_dir'
    for root, dirs, fnames in os.walk(input_dir):
        for i_fname in fnames:
            i_path = os.path.join(root, i_fname)
            yield i_path
<switch:english>
[english] The generator successfully yields comprehensive file paths by joining the current directory `root` with the specific filename `i_fname`. Note that it focuses exclusively on files and will not list directories themselves.
[english] What would happen if I wanted this function to also include all subdirectories in its output?
[english] To include directories, you can modify the loop to yield the 'root' path before iterating through filenames:
<switch:python>
[python] def paths_include_dirs(input_dir):
    for root, dirs, fnames in os.walk(input_dir):
        yield root # Yield the directory itself
        for i_fname in fnames:
            i_path = os.path.join(root, i_fname)
            yield i_path
