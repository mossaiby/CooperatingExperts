[english] How can I filter my database of atomic species to retrieve only those nuclei that have both an odd number of protons (Z) and an odd number of neutrons (N)?
[english] This function defines a method to isolate odd-odd nuclei by using the select function on the current object. It applies a lambda expression as a test, checking if both Z and N are odd numbers.
<switch:python>
[python] def odd_odd(self):
        """Selects odd-odd nuclei from the table:

        >>> Table('FRDM95').odd_odd
        Out[13]:
        Z   N
        9   9       1.21
            11      0.10
            13      3.08
            15      9.32
        ...
        """
        return self.select(lambda Z, N: (Z % 2) and (N % 2), name=self.name)

[english] The boolean logic in ``(Z % 2) and (N % 2)`` effectively filters out any nucleus where at least one of the proton or neutron counts is even.
