[english] I need a way to iterate through a dictionary where I can simultaneously transform keys and values, and selectively include items based on specific logic.
[english] We can implement this using a single dictionary comprehension that evaluates three main things: inclusion via `if_func`, key transformation with `key_func`, and value transformation with `value_func`.
<switch:python>
[python] def map_dict(d, key_func=None, value_func=None, if_func=None):
    '''
    :param dict d: dictionary
    :param func key_func: func which will run on key.
    :param func value_func: func which will run on values.
    '''
    key_func = key_func or (lambda k, v: k)
    value_func = value_func or (lambda k, v: v)
    if_func = if_func or (lambda k, v: True)
    return {
        key_func(*k_v): value_func(*k_v)
        for k_v in d.items() if if_func(*k_v)
    }
<switch:english>
[english] This function generalizes mapping for dictionaries by allowing custom functions to handle filtering, key modification, and value extraction.
[english] What happens if either `key_func` or `value_func` encounters an error while processing a specific item?
[english] The exception will propagate through the dictionary comprehension just like any other Python operation. You should wrap the call to it in a try/except block if you expect errors from your transformation functions.
