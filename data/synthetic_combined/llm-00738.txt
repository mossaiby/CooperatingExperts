[english] How can I calculate the central frequency for several coarse channels when given a continuous array of frequency bins?
[english] We first determine how many coarse channels there are, then reorganize the flat frequency array into groups. Finally, we compute the mean across each group to find the center frequency.
<switch:python>
[python] def get_centerfreqs(freqs,chan_per_coarse):
    '''
    Returns central frequency of each coarse channel

    Parameters
    ----------
    freqs : 1D Array (float)
        Frequency values for each bin of the spectrum
    chan_per_coarse: int
        Number of frequency bins per coarse channel
    '''

    num_coarse = freqs.size/chan_per_coarse
    freqs = np.reshape(freqs,(num_coarse,chan_per_coarse))
    return np.mean(freqs,axis=1)
