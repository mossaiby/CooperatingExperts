[english] How can I traverse my custom tree structure and set the name attribute to None for every internal node?
[english] To accomplish this, we will utilize a visitor pattern that systematically traverses the entire tree. We define the action (setting the name) and restrict its application using a predicate to only process nodes that are not leaves.
<switch:python>
[python] def remove_internal_names(self):
        """
        Set the name of all non-leaf nodes in the subtree to None.
        """
        self.visit(lambda n: setattr(n, 'name', None), lambda n: not n.is_leaf)
<switch:english>
[english] This approach efficiently targets all non-leaf structures by checking if `n` is not a leaf. Remember that this transformation modifies the structure in place, affecting how those internal nodes are referenced.
[english] What happens if an internal node already has its name attribute set to None?
[english] The function will simply execute setattr(n, 'name', None) again. Since it is a deterministic operation, running it multiple times on the same node has no adverse effect.
