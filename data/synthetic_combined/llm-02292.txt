[english] How can I ensure all my buffered metrics are sent over a network connection once, even if they were added incrementally?
[english] This `flush` method is designed to empty internal buffers and synchronously send them out.

[python] def flush(self):
        # type: () -> BatchClient
        """Send buffered metrics in batch requests"""

        address = self.remote_address
        while len(self._batches) > 0:
            self._socket.sendto(self._batches[0], address)
            self._batches.popleft()
        return self

[english] The function iteratively pops and sends each batch until the queue is empty, then returns `self` to allow method chaining.
