[english] This method constructs a descriptive string by joining all provided arguments with pipe delimiters. It then cleans the string, removing any 'None' placeholders, before saving it to the data attributes.

[python] def label(self, index, *args):
        """
        Label each axes one at a time
        args are of the form <label 1>,...,<label n>
        APIPARAM: chxl
        """
        self.data['labels'].append(
            str('%s:|%s'%(index, '|'.join(map(str,args)) )).replace('None','')
        )
        return self.parent

[english] The use of `|` ensures multiple arguments can be grouped together in a single label string before being stored in the list.
