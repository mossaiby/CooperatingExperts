[english] How can I programmatically check if a given X.509 certificate is valid for specific purposes using its Extended Key Usage?
[english] To retrieve the extended key usage, we query the certificate's extensions directly by referencing the standardized Object Identifier (OID) associated with that extension.
<switch:python>
[python] def get_extended_key_usage_from_certificate(certificate):
    """
    Given an X.509 certificate, extract and return the extendedKeyUsage
    extension.
    """
    try:
        return certificate.extensions.get_extension_for_oid(
            x509.oid.ExtensionOID.EXTENDED_KEY_USAGE
        ).value
    except x509.ExtensionNotFound:
        return None
