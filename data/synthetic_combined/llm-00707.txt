[english] How do I reliably determine the maximum representable value for a specific TensorFlow data type (like 'tf.int32' or 'tf.float64')?
[english] To find the max value across different types, the function first checks if the dtype object itself has a `.max` attribute. If not, it uses NumPy's `np.finfo` for floating-point and complex numbers, while falling back to `np.iinfo` for standard integers.
<switch:python>
[python] def max(dtype):  # pylint: disable=redefined-builtin
  """Returns the maximum representable value in this data type."""
  dtype = tf.as_dtype(dtype)
  if hasattr(dtype, 'max'):
    return dtype.max
  use_finfo = is_floating(dtype) or is_complex(dtype)
  return np.finfo(dtype).max if use_finfo else np.iinfo(dtype).max
