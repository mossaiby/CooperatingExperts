[english] How can I create a dictionary that maps each model parameter name to its specified default value and its currently assigned actual value?
[english] This function iterates through all available parameters in self.parms. For each one, it constructs a nested dictionary containing both the 'default' and 'actual' values before adding it to the main results dictionary.
<switch:python>
[python] def params(self):
        """
        Get the parameters and the actual/default values only.

        :returns: A dictionary of parameters used to build this model.
        """
        params = {}
        for p in self.parms:
            params[p] = {"default": self.parms[p]["default_value"],
                         "actual": self.parms[p]["actual_value"]}
        return params

[english] The function is highly dependent on self.parms having consistent keys like 'default_value' and 'actual_value'. If those sub-keys are ever missing, this code will raise a KeyError during execution.
[english] What if one of the parameters in self.parms does not have an 'actual_value'? Will that cause an error?
[english] Yes, accessing a missing key directly causes a KeyError. You can use the .get() method to provide fallback values instead.
<switch:python>
[python] def params(self):
    params = {}
    for p in self.parms:
        data = self.parms[p]
        params[p] = {"default": data.get("default_value"),
                     "actual": data.get("actual_value")}
    return params
