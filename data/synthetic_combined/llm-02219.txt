[english] I need a helper function to calculate and format event start and end datetimes based on configuration.
[english] This approach calculates the current time once, then returns a dictionary using `datetime.timedelta` to offset that time for both the 'start' and 'end' keys.
<switch:python>
[python] def create_event_datetimes(options, config):
    """
    Create event start and end datetimes.
    """

    now = datetime.datetime.now()

    return {
        "start": {
            "dateTime": (now + datetime.timedelta(minutes=int(config["start"]))).strftime(DT_FORMAT),
            "timeZone": options.timezone,
        },
        "end": {
            "dateTime": (now + datetime.timedelta(minutes=int(config["end"]))).strftime(DT_FORMAT),
            "timeZone": options.timezone,
        },
    }
