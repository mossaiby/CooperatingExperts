[english] How can I reliably convert a SQLAlchemy model instance into a simple Python dictionary using its field names as keys?
[english] We will iterate through the columns defined in the object's associated database table. For each column, we dynamically retrieve the attribute value and map it directly to a key in a new dictionary.
<switch:python>
[python] def alchemy_to_dict(obj):
    """
    Transforms a SQLAlchemy model instance into a dictionary
    """
    if not obj:
        return None
    d = {}
    for c in obj.__table__.columns:
        value = getattr(obj, c.name)
        if type(value) == datetime:
            value = value.isoformat()
        d[c.name] = value
    return d

[english] The code includes specific logic that checks if a retrieved value is a datetime object. If so, it converts it using ISO formatting before adding it to the resulting dictionary.

[english] The current function only processes direct column values and does not handle complex object nesting. To serialize relationships, you would need to implement recursive calls to this function.
