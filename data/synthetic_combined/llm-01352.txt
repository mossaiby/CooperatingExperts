[english] I need a way to iterate over any sequence and receive tuples indicating whether each item is the last one.
[english] This function converts the input into an iterator and uses a shifting mechanism (holding onto `prev`) to always lag by one element. This allows it to determine if the current state represents the final true item.
<switch:python>
[python] def iter_with_last(iterable):
    """
    :return: generator of tuples (isLastFlag, item)
    """
    # Ensure it's an iterator and get the first field
    iterable = iter(iterable)
    prev = next(iterable)
    for item in iterable:
        # Lag by one item so I know I'm not at the end
        yield False, prev
        prev = item
    # Last item
    yield True, prev
