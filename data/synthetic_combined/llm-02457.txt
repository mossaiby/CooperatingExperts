[english] I have a 16-bit unsigned integer I need to send over MQTT; how can I reliably convert this number into two separate bytes?
[english] The function first ensures the input is treated as an integer, then splits it into high and low bytes using bitwise shifting. By assigning these components to a bytearray in order, we achieve standard Big Endian network serialization.
<switch:python>
[python] def encode16Int(value):
    '''
    Encodes a 16 bit unsigned integer into MQTT format.
    Returns a bytearray
    '''
    value      = int(value)
    encoded    = bytearray(2)
    encoded[0] = value >> 8
    encoded[1] = value & 0xFF
    return encoded
