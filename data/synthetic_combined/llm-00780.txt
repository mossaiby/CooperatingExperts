[english] How can I safely remove a specific connection between two instances stored within a network structure?
[english] This function implements a safe removal strategy by first confirming that both target objects are present in the data structure before attempting any modification.

[python] def disconnect(self, instance, another_instance):
        '''
        Disconnect an *instance* from *another_instance*.
        '''
        if instance not in self:
            return False
        
        if another_instance not in self[instance]: 
            return False

        self[instance].remove(another_instance)
        return True
