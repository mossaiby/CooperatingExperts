[english] How can I validate that two dimensions correspond to an n-qubit channel in quantum computing?
[english] This function verifies two key properties of an n-qubit system: first, that the input and output tensor dimensions match exactly. Second, it ensures that the dimension is a perfect power of 2 using logarithms.

[python] def _check_nqubit_dim(input_dim, output_dim):
    """Return true if dims correspond to an n-qubit channel."""
    if input_dim != output_dim:
        raise QiskitError(
            'Not an n-qubit channel: input_dim' +
            ' ({}) != output_dim ({})'.format(input_dim, output_dim))
    num_qubits = int(np.log2(input_dim))
    if 2**num_qubits != input_dim:
        raise QiskitError('Not an n-qubit channel: input_dim != 2 ** n')
