[english] How can I write a function that retrieves a complete, structured list of all BigQuery datasets within a specified Google Cloud project?
[english] This function is designed to interface directly with the BigQuery API to fetch dataset metadata. It first determines which project ID to use by prioritizing an input parameter over its own internal stored value.
<switch:python>
[python] def get_datasets_list(self, project_id=None):
        """
        Method returns full list of BigQuery datasets in the current project

        .. seealso::
            For more information, see:
            https://cloud.google.com/bigquery/docs/reference/rest/v2/datasets/list

        :param project_id: Google Cloud Project for which you
            try to get all datasets
        :type project_id: str
        :return: datasets_list

            Example of returned datasets_list: ::

                   {
                      "kind":"bigquery#dataset",
                      "location":"US",
                      "id":"your-project:dataset_2_test",
                      "datasetReference":{
                         "projectId":"your-project",
                         "datasetId":"dataset_2_test"
                      }
                   },
                   {
                      "kind":"bigquery#dataset",
                      "location":"US",
                      "id":"your-project:dataset_1_test",
                      "datasetReference":{
                         "projectId":"your-project",
                         "datasetId":"dataset_1_test"
                      }
                   }
                ]
        """
        dataset_project_id = project_id if project_id else self.project_id

        try:
            datasets_list = self.service.datasets().list(
                projectId=dataset_project_id).execute(num_retries=self.num_retries)['datasets']
            self.log.info("Datasets List: %s", datasets_list)

        except HttpError as err:
            raise AirflowException(
                'BigQuery job failed. Error was: {}'.format(err.content))

        return datasets_list
<switch:english>
[english] It leverages the BigQuery service call to list all resources under a given project ID. Robust error handling is included via try/except blocks that convert API errors into specific Airflow exceptions.
[english] If `self.service` or `self.num_retries` are not initialized, will this function still execute successfully?
[english] No, the function relies heavily on these attributes being present upon instantiation to correctly perform the service call and manage retries. You would need initialization code like this:
<switch:python>
[python] class BigQueryHelper: 
    def __init__(self, project_id): 
        self.project_id = project_id
        self.service = self._initialize_bigquery_client()
        self.num_retries = 3
