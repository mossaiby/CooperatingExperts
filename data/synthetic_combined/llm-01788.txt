[english] How do I safely remove all traces of a defined variable from memory across various namespaces to prevent hidden references?
[english] This function is designed to ensure that when you delete a variable, no lingering references remain in other parts of the execution environment. It first validates the input and then decides whether to clear variables using their name or by finding matching object instances.
<switch:python>
[python] def del_var(self, varname, by_name=False):
        """Delete a variable from the various namespaces, so that, as
        far as possible, we're not keeping any hidden references to it.

        Parameters
        ----------
        varname : str
            The name of the variable to delete.
        by_name : bool
            If True, delete variables with the given name in each
            namespace. If False (default), find the variable in the user
            namespace, and delete references to it.
        """
        if varname in ('__builtin__', '__builtins__'):
            raise ValueError("Refusing to delete %s" % varname)

        ns_refs = self.all_ns_refs
        
        if by_name:                    # Delete by name
            for ns in ns_refs:
                try:
                    del ns[varname]
                except KeyError:
                    pass
        else:                         # Delete by object
            try:
                obj = self.user_ns[varname]
            except KeyError:
                raise NameError("name '%s' is not defined" % varname)
            # Also check in output history
            ns_refs.append(self.history_manager.output_hist)
            for ns in ns_refs:
                to_delete = [n for n, o in ns.iteritems() if o is obj]
                for name in to_delete:
                    del ns[name]

            # displayhook keeps extra references, but not in a dictionary
            for name in ('_', '__', '___'):
                if getattr(self.displayhook, name) is obj:
                    setattr(self.displayhook, name, None)
<switch:english>
[english] By traversing all tracked namespaces and checking for identity (`is obj`), it aggressively cleans up existing references. Note that relying on `by_name=False` is powerful but requires certainty about how the variable was referenced elsewhere.
[english] What happens if the object I am trying to delete is an immutable type, like a string, and multiple variables reference identical instances of that string?
[english] Because it uses identity checking (`is obj`), all namespaces holding a direct reference to that exact shared instance of the string will be cleaned up by this function.
