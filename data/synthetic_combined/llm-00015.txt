[english] How can I extract and appropriately format operational parameters from an instruction object into a list of strings suitable for use in a label?
[english] This function first verifies if the instruction has parameters defined. It includes logic to return None early if all parameters are NumPy ndarrays, as these types cannot be easily serialized.

[python] def params_for_label(instruction):
        """Get the params and format them to add them to a label. None if there
         are no params of if the params are numpy.ndarrays."""

        if not hasattr(instruction.op, 'params'):
            return None
        if all([isinstance(param, ndarray) for param in instruction.op.params]):
            return None

        ret = []
        for param in instruction.op.params:
            if isinstance(param, (sympy.Number, float)):
                ret.append('%.5g' % param)
            else:
                ret.append('%s' % param)
        return ret
