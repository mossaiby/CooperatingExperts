[english] How can I automatically wrap a piece of inner XML data inside a defined root element while ensuring it includes the required Azure namespace?
[english] This function achieves its goal by utilizing string concatenation and basic Python formatting to construct the necessary opening tag, inject the specified namespace attribute, append your internal XML content, and finally close the outer element.
<switch:python>
[python] def doc_from_xml(document_element_name, inner_xml,
                     xmlns='http://schemas.microsoft.com/windowsazure'):
        '''Wraps the specified xml in an xml root element with default azure
        namespaces'''
        xml = ''.join(['<', document_element_name,
                      ' xmlns="{0}">'.format(xmlns)])
        xml += inner_xml
        xml += ''.join(['</', document_element_name, '>'])
        return xml
