[english] How do I ensure my data loader shuffles the dataset at the end of every epoch?
[english] This method prepares an index array that points to all rows in your dataset. It initializes these indices and conditionally shuffles them if you have enabled shuffling.
<switch:python>
[python] def on_epoch_end(self) -> None:
        'Updates indexes after each epoch for shuffling'
        self.indexes = np.arange(self.nrows)
        if self.shuffle:
            np.random.shuffle(self.indexes)
<switch:english>
[english] By setting `self.indexes` to a random permutation, the subsequent iterations will draw data in a randomized order.
[english] What happens if I set `self.shuffle` to false?
[english] If `self.shuffle` is false, the function sets indexes but skips the shuffle step, meaning your data will be pulled in its original sequence for that epoch.
