[english] This function handles three distinct cases for the input value `net`. It determines if we should jump back to a starting point (if net is 0) or simply increment/decrement based on whether net is positive or negative.

[python] def get_next_and_prev(net):
    """Returns what the next and prev querystrings should be."""
    if net == 0:
        nxt = prev = 1
    elif net > 0:
        nxt = net + 1
        prev = -(net - 1)
    else:
        nxt = net + 1
        prev = abs(net) + 1
    return nxt, prev
