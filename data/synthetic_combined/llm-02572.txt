[english] I have an object and need a way to calculate its current nesting level based on surrounding specific container types like Templates or ParserFunctions.
[english] The function determines how many defined parent structures wrap the given item. It iterates over two key container types, 'Template' and 'ParserFunction', looking for spans that completely encompass the current object's boundaries.
<switch:python>
[python] def nesting_level(self) -> int:
        """Return the nesting level of self.

        The minimum nesting_level is 0. Being part of any Template or
        ParserFunction increases the level by one.
        """
        ss, se = self._span
        level = 0
        type_to_spans = self._type_to_spans
        for type_ in ('Template', 'ParserFunction'):
            spans = type_to_spans[type_]
            for s, e in spans[:bisect(spans, [ss + 1])]:
                if se <= e:
                    level += 1
        return level
