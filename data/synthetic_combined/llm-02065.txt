[english] How can I extract a unique list of all clinical gene symbols from the specified panels of a case object?
[english] This approach first gathers all panel IDs from the case object. Then, it uses a MongoDB aggregation pipeline to efficiently query the database and collect all associated gene symbols.
<switch:python>
[python] def clinical_symbols(self, case_obj):
        """Return all the clinical gene symbols for a case."""
        panel_ids = [panel['panel_id'] for panel in case_obj['panels']]
        query = self.panel_collection.aggregate([
            {'$match': {'_id': {'$in': panel_ids}}},
            {'$unwind': '$genes'},
            {'$group': {'_id': '$genes.symbol'}}
        ])
        return set(item['_id'] for item in query)
<switch:english>
[english] The aggregation pipeline handles the deduplication automatically by using a group stage, resulting in a unique set of symbols.
[english] What is the function likely to return if none of the provided panel IDs are found in the database?
[english] If no matching panels are found, the `$match` filter returns an empty cursor. The final function call `set(item['_id'] for item in query)` will then correctly return an empty set.
