[english] I need a way to set up my application's logging system so that messages appear both in the operating system's syslog and on the console, depending on how verbose I want to be.
[english] This function automates log setup by initializing two types of handlers: one designed to forward logs directly to the system logger, and an optional second handler for printing output to standard streams like the terminal. It allows you to control how detailed the logging is using 'quiet' and 'debug' flags.
<switch:python>
[python] def init_log( quiet, debug ):
   '''
   setup system logging to desired verbosity.
   '''
   from logging.handlers import SysLogHandler
   fmt = logging.Formatter( os.path.basename(sys.argv[0]) +
         ".%(name)s %(levelname)s - %(message)s")
   facility = SysLogHandler.LOG_DAEMON
   syslog = SysLogHandler(address='/dev/log',facility=facility)
   syslog.setFormatter( fmt )
   log.addHandler(syslog)
   if not quiet:
      console = logging.StreamHandler()
      console.setFormatter( fmt )
      log.addHandler(console)
      log.setLevel(logging.INFO)
      if debug:
         log.setLevel(logging.DEBUG)
