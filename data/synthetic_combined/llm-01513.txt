[english] How can I programmatically ensure my logger has a valid connection to ConnectorDB, recreating it if necessary?
[english] This method implements a lazy-loading pattern for database connections. It checks if the private `__cdb` attribute is unset and only creates a new `ConnectorDB` object when first requested or after a prior session ends.

[python] def connectordb(self):
        """Returns the ConnectorDB object that the logger uses. Raises an error if Logger isn't able to connect"""
        if self.__cdb is None:
            logging.debug("Logger: Connecting to " + self.serverurl)
            self.__cdb = ConnectorDB(self.apikey, url=self.serverurl)
        return self.__cdb
