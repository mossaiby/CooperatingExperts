[english] I need a way to temporarily update my system's environment variables so that python module lookups include all directories currently in sys.path.
[english] This function captures a copy of the existing operating system environment and then specifically overrides the 'PYTHONPATH'. By joining sys.path elements, it ensures your current working directories are made available for imports within this new map.
<switch:python>
[python] def _get_env():
    """Extracts the environment PYTHONPATH and appends the current sys.path to
    those."""
    env = dict(os.environ)
    env["PYTHONPATH"] = os.pathsep.join(sys.path)
    return env
