[english] How do I start my application using an asynchronous network framework while ensuring it handles graceful shutdown signals?
[english] To ensure reliable background service operation, we need to hook into OS interrupt signals so your app can exit cleanly when requested. We then initialize the reactive event loop.

    signal.signal(signal.SIGTERM, _handle_term_signal)
    signal.signal(signal.SIGINT, _handle_term_signal)

[english] `handle_term_signal` should ideally initiate asynchronous cleanup tasks like closing database connections or flushing stored buffers before allowing the main loop to stop. If it only logs a message, you might want to add logic similar to this: `tornado.ioloop.IOLoop.current().stop()`.
