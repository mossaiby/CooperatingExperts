[english] To isolate a single record in a sequential file, we first need to calculate the precise starting byte offset based on the record's 1-based index. This is done by multiplying the index by the record length K.

[python] def read_record(self, n):
        """Return record `n` as 1,024 bytes; records are indexed from 1."""
        self.file.seek(n * K - K)
        return self.file.read(K)
