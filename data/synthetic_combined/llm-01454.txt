[english] I need a robust method to filter my features so that only those which exceed a minimum average importance (threshold) across a specific set of study IDs are retained.
[english] This function calculates the aggregate value for each feature based on the provided IDs and then filters out any readings that fall below your defined threshold. It handles both returning just the names or the full data Series.
<switch:python>
[python] def get_features_by_ids(self, ids=None, threshold=0.0001, func=np.mean,
                            get_weights=False):
        ''' Returns features for which the mean loading across all specified
        studies (in ids) is >= threshold. '''
        weights = self.data.ix[ids].apply(func, 0)
        above_thresh = weights[weights >= threshold]
        return above_thresh if get_weights else list(above_thresh.index)
