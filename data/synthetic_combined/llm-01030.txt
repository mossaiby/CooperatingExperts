[english] I have a string like 'ampl1=0.5, ampl2=1.2' and need to convert it into a dictionary where the keys are names and values are floating-point numbers.
[english] To process this, we initialize an empty dictionary and split the input string by commas. Then, for each resulting segment, we split on the equals sign to separate the name from its value, stripping any extra whitespace before saving it to our gains dictionary.

[python] def specific_gains(string):
    """Convert string with gains of individual amplification elements to dict"""
    if not string:
        return {}

    gains = {}
    for gain in string.split(','):
        amp_name, value = gain.split('=')
        gains[amp_name.strip()] = float(value.strip())
    return gains

[english] The use of `.strip()` is key here as it ensures that any surrounding whitespace from the commas or equals signs doesn't interfere with setting correct dictionary keys and converting values to floats.
